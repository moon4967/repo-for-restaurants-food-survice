
# Restaurant Food Ordering System - Surveys

>  Note: Results below are a realistic sample/template to demonstrate format and analysis style. Replace with results from a survey you actually distribute before final submission.

## Executive Summary
A customer survey was conducted to quantify ordering preferences and frustrations at a broader scale than interviews alone, validating findings from the interview phase.

## Objective
To gather measurable, broad-based customer feedback on ordering experience and feature preferences.

## Methodology
| Parameter | Detail |
|---|---|
| Audience | Customers who ordered from a restaurant in the last 3 months |
| Sample size | 20 respondents |
| Distribution | Google Form shared via social media/messaging groups |
| Duration | 1 week |

## Survey Questions
| Question ID | Question | Type |
|---|---|---|
| Q-01 | How often do you order food per week? | Multiple choice |
| Q-02 | How do you usually place your order? | Multiple choice |
| Q-03 | Have you experienced an incorrect order in the last 3 months? | Yes/No |
| Q-04 | How important is real-time order tracking to you? | 1-5 scale |
| Q-05 | Would you use a restaurant's own ordering system instead of a third-party app? | Yes/No/Maybe |
| Q-06 | What frustrates you most about the current ordering process? | Open-ended |

## Results Summary
| Question ID | Result |
|---|---|
| Q-01 | 60% order 1-2x/week, 25% order 3-5x/week, 15% rarely |
| Q-02 | 45% phone/in-person, 35% third-party app, 20% other |
| Q-03 | 65% Yes, 35% No |
| Q-04 | Average rating 4.2 / 5 |
| Q-05 | 70% Yes, 20% Maybe, 10% No |
| Q-06 | Common themes: unclear ready times, wrong items delivered, busy phone lines, third-party fees |

## Key Findings
| Finding ID | Finding | Data Support | Implication |
|---|---|---|---|
| F-01 | Majority of customers have experienced an incorrect order | 65% (Q-03) | Validates order accuracy as a top priority requirement |
| F-02 | Real-time tracking is highly valued | 4.2/5 average (Q-04) | Validates status-tracking as a core feature |
| F-03 | Strong preference for restaurant-owned ordering system | 70% (Q-05) | Validates core product value proposition |

## Conclusion
Survey results corroborate interview findings: customers want accurate orders, real-time status visibility, and an alternative to high-fee third-party platforms. These findings directly carry forward into PRD user stories and acceptance criteria.
