# Restaurant Food Ordering System - Stakeholder Analysis

## Executive Summary
This document identifies all individuals and groups impacted by or able to influence the Restaurant Food Ordering System, along with their interests, influence level, and expectations, to ensure the system is designed to meet their needs.

## Stakeholder Register
| Stakeholder ID | Stakeholder | Category | Role |
|---|---|---|---|
| SH-01 | Customers | External / End user | Place orders, pay, track status |
| SH-02 | Restaurant Owner/Manager | Internal / Sponsor | Defines requirements, approves features |
| SH-03 | Cashier/Front-of-house Staff | Internal / End user | Confirms orders, handles payment |
| SH-04 | Kitchen Staff | Internal / End user | Views and updates order preparation status |
| SH-05 | Delivery Staff | Internal / End user | Manages delivery order status |
| SH-06 | System Admin | Internal / End user | Manages menu, staff accounts, configuration |
| SH-07 | Development Team | Internal | Designs, builds, tests the system |
| SH-08 | Course Instructor/Evaluator | External | Assesses deliverables and quality |

## Stakeholder Interest & Influence
| Stakeholder ID | Interest Level | Influence Level | Key Expectations |
|---|---|---|---|
| SH-01 | High | Low-Medium | Accurate orders, real-time tracking, secure payment |
| SH-02 | Very High | High | Error reduction, efficiency, sales visibility |
| SH-03 | High | Medium | Simple interface, fast confirmation |
| SH-04 | High | Medium | Clear, real-time order details |
| SH-05 | Medium | Low-Medium | Clear delivery details, status updates |
| SH-06 | High | Medium-High | Easy menu/staff management, reliability |
| SH-07 | Very High | High | Clear requirements, feasible scope |
| SH-08 | Medium | High | Well-documented, demonstrable understanding |

## Power-Interest Matrix
| Quadrant | Stakeholders | Engagement Strategy |
|---|---|---|
| Manage Closely (High Power, High Interest) | SH-02, SH-07 | Frequent reviews, direct involvement in decisions |
| Keep Informed (High Interest, Lower Power) | SH-01, SH-03, SH-04 | Regular updates, feedback channels |
| Keep Satisfied (High Power, Lower Interest) | SH-06, SH-08 | Periodic reporting, milestone reviews |
| Monitor (Lower Power, Lower Interest) | SH-05 | Minimal effort, monitor for changes |
